# Architecture Decision Records

Each ADR records one significant, hard-to-reverse decision: the context that forced
it, the decision itself, and the consequences (including what it costs). ADRs are
immutable once accepted — a later decision supersedes an earlier one with a new
record rather than editing history.

Status values: **Proposed** · **Accepted** · **Superseded by ADR-NNN**.
